## ğŸ“ How to Add Credits

When you add new assets to the game:

1. **Identify the license** - Check the source
2. **Add attribution** - Update this file with proper credit
3. **Follow license terms** - Some require specific attribution format
4. **Keep links working** - Verify source URLs are accessible

### Common License Types

- **CC0** - Public domain, no attribution required (but nice to include)
- **CC-BY** - Attribution required
- **CC-BY-SA** - Attribution + Share-alike
- **MIT** - Very permissive, attribution in code
- **Apache 2.0** - Similar to MIT with patent grant

### Attribution Template

```markdown
### [Asset Category]

**[Asset Name]**
- **Author**: [Name]
- **Source**: [URL]
- **License**: [License Type]
- **Modifications**: [None / List changes]
- **Date Added**: [YYYY-MM-DD]
```
