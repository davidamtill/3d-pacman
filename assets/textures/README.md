# Textures

Place your texture files here (.png, .jpg, .webp).

## Suggested Textures

- `spaceship-diffuse.png` - Player ship color map
- `spaceship-normal.png` - Normal map for detail
- `spaceship-metallic.png` - Metallic/roughness map
- `enemy-diffuse.png` - Enemy color map
- `particle.png` - For explosion effects
- `star.png` - Enhanced starfield particles

### Skybox (optional)
- `skybox-px.png` - Positive X
- `skybox-nx.png` - Negative X
- `skybox-py.png` - Positive Y
- `skybox-ny.png` - Negative Y
- `skybox-pz.png` - Positive Z
- `skybox-nz.png` - Negative Z

## Current Implementation

The game uses:
- Solid colors for materials
- Procedural starfield (white points)
- No texture mapping

You can enhance the visuals by adding textures and updating the material definitions in the game code.
